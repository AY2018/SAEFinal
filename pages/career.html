Jobs I would like to do when I graduate

1. Front end web Developer

Skills needed

  • Html
  • CSS
  • CSS Frameworks/Libraries (Bootstrap, Tailwind CSS, Sass)
  • JavaScript
  • JavaScript libraries and frameworks (React, Vue)
  • Databases (SQL, MySQL)

Front End Development relates to everything the user can see and interact with. For example, everything on this web site from the logo, the menu, to the images, videos, animations and layout of the page are managed by the front end developer, or in this case, me. To summarize, the front end developer is in charge of the look and feel of the website, on top of making sure that it looks good on all devices (phone, tablet, laptop).

I have loved building legos ever since I was young and I felt the same satisfaction I got finishing a lego when I built my first website in HTML and CSS. Ever since then, it became a hobby of mine to just build websites in my free time. Thus it would be incredible if I could make it my job one day to build websites to help other people grow their business.

2. Back end web Developer

Skills needed

  • Python
  • (or) Java
  • (or) Node
  • (or) PHP

Back End Development refers to the "behind the scenes of the website", the part of the application that the user does not see. Back end developers focus on the logic of the site, creating servers, and working with databases and APIs (Application Programming Interfaces).

For example, the content and layout of the form in this page's footer is visible on the client side, but when the users submit their information it gets processed on the server side.

I haven't started learning any of the back end languages yet, but I'm looking forward to working on a more "technical" aspect of web development. I have been good at maths since young and do generally well when it comes to logic so this might be an interesting job for me.

3. Full stack Engineer

Skills needed

  • All of the skills mentioned above

A Full Stack Developer is someone who works on both the front and back end of a web application. Working on both sides, he must have a good mastery of all the languages used whether in front end or back end. A Full Stack Developer also needs good problem solving skills as well as good debugging skills.

Full Stack Development looks like a good option further down my path as a Web Developer. I don't know for now if I will start as a Front end or Back end developer, but full stack development, being the combination of both, is an interesting career choice a few years after I start working.

4. Web Designer

Skills needed

  • Html
  • CSS
  • CSS Frameworks/Libraries (Bootstrap, Tailwind CSS, Sass)
  • JavaScript
  • Figma
  • Photoshop

If a Front End Developer is responsible for building a website, then the Web Designer is responsible for designing the layout, visual appearance and the usability of a website. A good web designer needs to have both the technical skills (to propose the right design and not something that the programmer can't build) and the graphic skills.

A Web designer needs to be able to visualize how the web application will look like and how it will function (the code behind it).

I have now built a good amount of websites and I quite enjoyed thinking about how to structure them, how to choose the best colors to stay aligned with the brand's message and how to rearrange the elements depending on the device. Web Designer is therefore a possible alternative to my three preferred jobs that are more "programming oriented".
